# Quote Generator

This simple web application allows users to generate random quotes using the [Quotable API](http://api.quotable.io/). It is built using HTML, CSS (with Bootstrap), and JavaScript.

## Features

- Click the button to generate a random quote.
- The application fetches data from the Quotable API to provide a diverse set of quotes.

## Usage

1. **Clone the repository:**

   ```bash
   git clone https://github.com/MickaelGP/Quote_Generator.git

2. **Open the project folder:**
   - cd Quote_Generator
   - Open the index.html file in your web browser.
   - Click the button to generate a random quote.
